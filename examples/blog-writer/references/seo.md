# SEO & formatting

- Title: ≤ 60 characters, includes the core benefit or keyword.
- Excerpt / meta description: 1–2 sentences, ≤ 160 characters, compelling.
- Use descriptive H2/H3 headings that include natural keywords.
- Prefer semantic HTML: `<h2>`, `<h3>`, `<p>`, `<ul>`, `<ol>`, `<blockquote>`,
  `<a>`. No inline styles, no `<html>/<body>` wrappers.
- Tags: 2–4 lowercase topical tags (e.g. "cash flow", "budgeting", "invoicing").
- Link out to credible sources with descriptive anchor text.
